# El Pollo Loco

Welcome to **El Pollo Loco**, an exciting 2D jump-and-run browser game! Take on the role of Pepe as you run through the Mexican desert, fighting off evil chickens and gathering enough salsa bottles to defeat the giant boss chicken.

## 🚀 How to Run
There are two simple ways to run and play the game:
1. **Directly in Browser:** Simply double-click the `index.html` file to open it in your default web browser.
2. **VS Code Live Server (Recommended):** If you are using Visual Studio Code, right-click on `index.html` and select **"Open with Live Server"**.

No extra installation or software is required!

## ✨ Features
* **Two Action-Packed Levels:** Start with an easy path in Level 1 and experience a longer, much harder journey in Level 2!
* **Dynamic Enemies:** Battle against normal chickens, small fast chickens, and a vicious Endboss.
* **Collectibles:** 
  * Collect **Coins** to slowly regen your health.
  * Collect **Salsa Bottles** to throw at your enemies.
* **Custom Mechanics:** Jump on top of regular chickens to defeat them, or throw bottles to reduce the Endboss's health!
* **Responsive Design:** Supports mouse, keyboard, and touch controls for mobile devices.

## 🕹️ Controls
* **Arrow Left / Right:** Walk left and right
* **Arrow Up / Spacebar:** Jump
* **'D' Key:** Throw a Salsa Bottle

*(If you are playing on a mobile device, simply use the on-screen touch buttons.)*

## 🛠️ Built With
* HTML5 Canvas
* CSS3
* Vanilla JavaScript (Object-Oriented Programming)

## 📌 Game Story
Pepe's village has been overrun by a crazed gang of chickens controlled by a giant Endboss. It's up to you to navigate the perilous desert, collect spicy salsa bottles, and throw them at the Endboss to reclaim the village! Only those with the best jumping skills and aim can save the day.
